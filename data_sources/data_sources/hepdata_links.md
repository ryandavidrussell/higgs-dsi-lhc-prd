## Public Data Sources (HEPData)

All numerical values used in the analysis scripts correspond to the following
publicly available datasets.

---

### ATLAS ggF H → WW*
- HEPData record: ins1444991
- https://www.hepdata.net/record/ins1444991

### CMS VBF H → γγ
- HEPData record: ins2142341
- https://www.hepdata.net/record/ins2142341

### ATLAS ggF H → γγ
- HEPData record: ins1674946
- https://www.hepdata.net/record/ins1674946

### ATLAS ggF H → ZZ*
- HEPData record: ins1615206
- https://www.hepdata.net/record/ins1615206

---

Numerical values in this repository reproduce the central values and
uncertainties reported in these tables, with luminosity rescaling noted
explicitly in the analysis scripts.
